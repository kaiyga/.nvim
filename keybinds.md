Bar Buffer Navigation:
    <A-,>: Go to previous buffer
    <A-.>: Go to next buffer
    <A-<>: Move current buffer left
    <A->>: Move current buffer right
    <A-1> to <A-9>: Go to buffer 1 to 9
    <A-0>: Go to last buffer
    <A-c>: Close current buffer

Completion:
    <C-b>: Scroll documentation up
    <C-f>: Scroll documentation down
    <C-Space>: Trigger completion menu
    <C-e>: Abort completion
    <CR>: Confirm selected completion
    <C-k>: Move up in completion menu
Git:
    <leader>gp: Preview current hunk
    <leader>gt: Toggle current line blame

LSP:
    K: Show documentation for current symbol
    <leader>gd: Go to definition
    <leader>gr: Show references
    <leader>ca: Show code actions
    <leader>cp: Use Formatter

File Manager:
    <C-n>: Open file manager
    <leader>bf: Open buffers in file manager

Window Navigation:
    <C-h>, <C-j>, <C-k>, <C-l>: Navigate to left, down, up, right

Miscellaneous:
    -: Toggle file manager float
    <C-p>: Open Telescope to find files
    <leader>fg: Open Telescope for live grep
    <leader><leader>: Open Telescope for recent files
    <c-\>: Toggle terminal
    <leader>t to <leader>g: Run various tests
